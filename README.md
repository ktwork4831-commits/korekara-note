# これからノート

これからの人生を安心して楽しむために、大切な情報をシンプルに整理するWebアプリです。

## 機能

- 端末内への自動保存（localStorage）
- 項目ごとの共有オン・オフ
- 共有前のプレビュー
- スマートフォンの共有機能／クリップボードコピー
- 共有対象だけをCSVでダウンロード
- レスポンシブ表示

## 起動

`index.html` をブラウザで開くだけで利用できます。Netlifyではフォルダをそのままデプロイできます。

## 注意

暗証番号やパスワードそのものは入力せず、保管場所のみを記録してください。現在の保存先は利用端末のブラウザ内です。
